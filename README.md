# 📌 Project Overview

Dataset Link = https://www.kaggle.com/datasets/surajsingh83/blinkit-grocery-data

This project focuses on building an interactive *Blinkit Sales Dashboard* using *Power BI* to analyze sales performance, customer purchasing behavior, outlet performance, and product trends.

The dashboard helps in understanding key business metrics and provides meaningful insights through powerful visualizations and data analytics techniques.


# 🎯 Objectives

- Perform data cleaning and preprocessing  
- Analyze Blinkit sales data  
- Create interactive Power BI dashboards  
- Visualize sales and outlet performance  
- Identify top-performing products and categories  
- Generate business insights using data analytics
